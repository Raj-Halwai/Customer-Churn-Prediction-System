# Machine Learning Notebook

This folder contains the Jupyter Notebook used for the Customer Churn Prediction System.

## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building
6. Model Evaluation
7. Business Insights

- Random Forest Classifier
- XGBoost Classifier

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- XGBoost

The notebook demonstrates the complete end-to-end machine learning workflow used to predict customer churn.
